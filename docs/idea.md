# Top 5 de todo

## 1 descripcion de la aplicacion
En este proyecto los usuarios pueden crear listas personales de "Top 5" sobre cualquier tema: peliculas, videojuegos, canciones, etc.
Cada lista tiene 5 posiciones, esto hace que el usuario deba ordenarlos por prioridad y elegir lo mas importante

## 2 Problema que busca resolver 
Las personas tienen gustos y preferencias, quieren compartirlos y buscar quien esta de acuerdo o en desacuerdo con sus gustos, estas preferencias por lo general se quieren compartir o recordar, pero muchas apps de listas y rankings son demasiado complejas o permiten listas infinitas.
El formato "Top 5" es simple, rapido y divertido, y se centra en lo esencial.

## 3 Usuario objetivo 
- Personas que quieren compartir gustos con amigos.
- Personas que disfrutan creando rankings (películas, música, videojuegos…).
- Cualquier persona que quiera guardar sus “Top 5” personales.

## 4 Funcionalidades principales 
- Crear una lista Top 5 con título y categoría.
- Añadir exactamente 5 elementos a la lista.
- Editar y eliminar listas.
- Guardar listas en LocalStorage.
- API REST que devuelve listas de ejemplo.
- Vista de detalle de cada lista.

## 5 Funcionalidades opcionales
- Compartir listas mediante un enlace.
- Modo oscuro.
- Subir imágenes para cada elemento del Top 5.
- Ordenar elementos arrastrando (drag & drop).
- Guardar listas en el backend Express en vez de LocalStorage.
- Sistema de “likes” para listas populares.

## 6 Mejoras futuras
- Autenticación real (login/registro).
- Base de datos real (PostgreSQL o MongoDB).
- Perfil de usuario con sus listas públicas.
- Comentarios en listas.
- Exportar listas como imagen para compartir en redes.
- App móvil con React Native.
